
# Hotel Management System

The Hotel Management System is a simple Java application that allows users to manage hotel bookings, customer information, room allocation, and food orders. It provides a command-line interface for performing various operations related to hotel management.

## Features

The Hotel Management System offers the following features:

- Book a room: Allows the user to book a room by providing customer details such as name, address, phone number, and number of days of stay.
- Customer Information: Displays the information of all the customers who have booked a room.
- Rooms Allotted: Shows the list of rooms that have been allocated to customers.
- Search for customer: Allows the user to search for a customer by name and displays their details.
- Edit Customer Details: Enables the user to edit the address and phone number of a customer.
- Order Food from Restaurant: Provides options to order breakfast, lunch, or dinner from the hotel restaurant.
- Total Bill/Payment of the Customer: Displays the total bill and payment details of a customer.
- Exit: Terminates the program and exits the application.

## File Structure

The project consists of the following files:

- `HotelManagementSystem.java`: Contains the main class with the application entry point.
- `Hotel.java`: Defines the Hotel class with methods for different hotel management operations.
- `record.data`: A data file used to store customer records.
- `temp.data`: A temporary file used during customer details editing.

## Notes

- The application uses text files (`record.data` and `temp.data`) to store and retrieve customer records. Ensure that these files have read and write permissions.
- The program logic for certain operations, such as food ordering and billing, is not implemented in the provided code. You can extend the application by adding the necessary functionality.
